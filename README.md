# Portfolio Book

A personal portfolio website built as an animated interactive "portfolio book" that showcases profile details, skills, education, projects, internships, and certificates.

## Overview

This project uses HTML, CSS, and JavaScript to create a 3D book-style portfolio layout. The homepage displays a profile cover, and users can turn pages to view skills, education, projects, and experience.

## Features

- Animated page-turning book interface
- Profile section with photo, title, bio, and social links
- Skills and technologies overview
- Education timeline
- Project highlights with links to GitHub repositories
- Internship and certificate details
- Download CV button
- "Contact Me" button to flip all pages to the end

## Files

- `index.html` — main portfolio page markup
- `style.css` — page styling, layout, and book animation
- `script.js` — JavaScript page-turn interaction and opening animation
- `image.jpeg` — profile image used in the portfolio
- `Resume_Nandini Kulshrestha.pdf` — downloadable resume asset

## Technologies

- HTML5
- CSS3
- JavaScript
- Boxicons icon library

## How to Run

1. Open `index.html` in a web browser.
2. Use the arrow buttons on the book pages to flip forward and backward.
3. Click the social icons to visit external profiles.
4. Click "Download CV" to open the resume PDF.

## Notes

- The interface is styled for a desktop viewport and uses 3D transform animations.
- The page-turn effects are implemented by toggling CSS classes in `script.js`.
